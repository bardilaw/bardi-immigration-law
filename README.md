# Bardi Immigration Law

Website and online presence for Bardi Immigration Law — an independent immigration law firm.

## Overview

This repository contains the source code and assets for building the Bardi Immigration Law website, including brand identity, core messaging, and digital presence.

## Goals

- Establish brand identity and visual guidelines
- Build a professional law firm website
- Create a findable online presence (Google, social media, etc.)
- Support client acquisition and communication
